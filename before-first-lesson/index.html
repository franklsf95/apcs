课前准备

Feb 19th, 2013

请在上第一节课前准备好以下物品：

便携式计算机

  • 电池电量应能供2个小时以上的正常使用；
  • 系统应能够正常运行；
    • 注：本课程不包含修电脑项目；
  • 应运行 Windows 或 Mac OS X 操作系统；
    • 注：如果你使用 Linux，我相信你不用来上这门课；

  • 应已遵循以下步骤安装了 Eclipse IDE for Java Developers（图标如右）：

    1. 访问 Eclipse Downloads
    2. 单击 Eclipse IDE for Java Developers 下载（那个下载了大约180多万次的）
      • Windows 用户去看一眼你的C盘，有“Program Files (x86)”这个文件夹的，下载64位版本；
      • 如果只有一个“Program Files”，下载32位版本；
      • Mac OS X 用户下载64位版本；
    3. 下载完成后解压缩到eclipse目录，启动 Ecplise；
    4. 经过一番启动，如果你进入了一个欢迎界面写道“Welcome to the Eclipse IDE for Java Developers”，说明安装成功。
