Assembly Language


    MOV AL, 1h        ; Load AL with immediate value 1
    MOV CL, 2h        ; Load CL with immediate value 2
    MOV DL, 61h       ; Load DL with immediate value 97
    MOV EAX, [EBX]    ; Move the 4 bytes in EBX into EAX
    MOV [ESI+EAX], CL ; Move the contents of CL into ESI+EAX

Package

A Package is literally a package of pre-written Java codes that can be imported and reused.

Import a package like this:


import somepackage.*;
import somepackage.subpackage.*;
import somepackage.ClassName;

Class

A Java program must have at least one class, which must contain the main method.


    import package1.*;
    public class MyFirstClass
    {
        public static void main(String[] args)
        {
            //Codes goes here
        }
    }

A Few Concepts

  1. Codes ⊂ Methods ⊂ Classes
  2. The main method
  3. Reserved words: import, public, class, static, void
